# 🛒 BigBasket Sales Analysis Dashboard

## 📊 Project Overview

This project analyzes BigBasket product data to understand category performance, pricing strategy, and discount trends in the Indian e-commerce market.

---

## 🛠 Tools Used

* Power BI
* Excel / CSV

---

## 📈 Key Insights

* Beauty & Hygiene category generates highest sales
* Kitchen & Grocery categories have moderate performance
* Discounts vary significantly across categories
* Top products contribute major revenue
